账号注册

首先你需要

注册

一个 github 账号，如图所示，设置账号密码和邮箱，其它都可以使用默认设置。

https://xue.cn/books/github-quick-start/imgs/register.png

安装 git

安装 homebrew

安装 git

配置 git

配置一下全局用户名和全局邮箱

配置 ssh

git 连接 github 的时候会用到 ssh 。我们需要配置一下 ssh。

使用 ssh-keygen 命令来生成私钥和公钥。

$ ssh-keygen

https://xue.cn/books/github-quick-start/imgs/keygen.png

将公钥配置到 github 的

SSH keys

中

将输入的内容复制到 Key 中,Title 可以自定义一个名字。

https://xue.cn/books/github-quick-start/imgs/addsshkey.png

附：ssh-keygen 有两个常用的参数 -t 和 -C ，其中 -t 参数用来指定加密的算法默认值为 rsa 。-C 参数为注释,默认为当前用户名@本地主机名。

创建 github 新仓库

在创建仓库页面中输入自定义的仓库名然后点击 Create repository 就可以了。
